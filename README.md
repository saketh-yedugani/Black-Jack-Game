🃏 Black Jack Game

A simple and interactive Blackjack (21) game built using JavaScript. This project demonstrates core programming concepts such as game logic, condition handling, and DOM manipulation.

🚀 Features

🎴 Random card generation

➕ Dynamic score calculation

🧠 Game state handling (win / lose / continue)

💬 Real-time messages for user feedback

🟢 “Start Game” and “New Card” functionality

🏆 Blackjack detection (sum = 21)

🛠️ Tech Stack

JavaScript (Game logic)

HTML (Structure)

CSS (Styling)

🎮 How to Play

Click Start Game to begin.

You will be dealt two cards.

Click New Card to draw another card.

Try to get as close to 21 as possible:

If your sum < 21 → You can draw another card

If your sum = 21 → 🎉 Blackjack! You win

If your sum > 21 → ❌ You lose
